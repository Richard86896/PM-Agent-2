# 主题风格规范

生成页面时，用户可指定主题名称。未指定时使用**橙白经典**（默认）。

主题通过 Ant Design 5 的 `ConfigProvider` 注入，在 `App.jsx` 的根节点配置 `theme` 属性。

---

## 主题一览

| 主题名 | 适用场景 | 主色 | 侧边栏风格 |
|--------|---------|------|-----------|
| 橙白经典 | 通用供应链后台（默认） | `#FF6B00` | 白底深字 |
| 蓝色科技 | 数据平台、智能仓储、监控看板 | `#1677FF` | 深蓝底白字 |
| 暗黑专业 | 大屏驾驶舱、数据分析 | `#4096FF` | 黑底白字 |
| 绿色清简 | 财务结算、合规审批 | `#00B96B` | 白底深字 |
| 紫色商务 | 高端 B 端演示、采购决策 | `#722ED1` | 深紫底白字 |

---

## 各主题配置

### 橙白经典（默认）

**定位**：橙白撞色，活力感强，适合供应链操作类页面。

```jsx
// App.jsx ConfigProvider 配置
theme={{
  token: {
    colorPrimary: '#FF6B00',
    borderRadius: 6,
    colorBgContainer: '#ffffff',
    colorBgLayout: '#f5f5f5',
  }
}}
```

**侧边栏**：
- 背景色：`#ffffff`
- 菜单文字：`#333333`
- 选中高亮：`#FF6B00`
- 系统标题区背景：`#FF6B00`，文字白色

**页面内容区**：
- 背景：`#f5f5f5`
- 卡片/表格背景：`#ffffff`
- 标签栏背景：`#ffffff`，激活标签底部橙色下划线

---

### 蓝色科技

**定位**：稳重专业，数字感强，适合数据看板、智能调度类页面。

```jsx
theme={{
  token: {
    colorPrimary: '#1677FF',
    borderRadius: 4,
    colorBgContainer: '#ffffff',
    colorBgLayout: '#f0f5ff',
  }
}}
```

**侧边栏**：
- 背景色：`#001529`（antd 深蓝）
- 菜单文字：`rgba(255,255,255,0.65)`
- 选中高亮：`#1677FF`
- 系统标题区背景：`#002140`，文字白色

**页面内容区**：
- 背景：`#f0f5ff`
- 卡片/表格背景：`#ffffff`
- 标签栏背景：`#ffffff`，激活标签蓝色下划线

**补充细节**：
- 统计卡片使用蓝色渐变背景（`linear-gradient(135deg, #1677FF 0%, #69b1ff 100%)`）
- 图标颜色统一用蓝色系

---

### 暗黑专业

**定位**：暗色沉浸，高对比，适合大屏展示、数据驾驶舱、长时间盯屏场景。

```jsx
theme={{
  algorithm: theme.darkAlgorithm,  // 需要从 antd 导入 theme
  token: {
    colorPrimary: '#4096FF',
    borderRadius: 6,
    colorBgContainer: '#1f1f1f',
    colorBgLayout: '#141414',
  }
}}
```

> 注意：需要在文件顶部导入 `import { theme } from 'antd'`

**侧边栏**：
- 背景色：`#0d0d0d`
- 菜单文字：`rgba(255,255,255,0.75)`
- 选中高亮：`#4096FF`
- 系统标题区背景：`#000000`，文字 `#4096FF`

**页面内容区**：
- 背景：`#141414`
- 卡片/表格背景：`#1f1f1f`
- 标签栏背景：`#1f1f1f`，激活标签蓝色下划线
- 分割线颜色：`#333333`

**补充细节**：
- 统计数字用亮色（`#4096FF` 或 `#52c41a`）形成对比
- 边框颜色统一用 `#333333`

---

## 场景匹配规则（自动推荐依据）

当用户未指定主题时，根据项目描述中的关键词自动推荐：

| 关键词 / 场景特征 | 推荐主题 | 推荐理由 |
|-----------------|---------|---------|
| 供应链、采购、库存、配送、订单管理、门店运营 | 橙白经典 | 通用供应链操作场景，活力感强 |
| 数据平台、智能调度、仓储监控、IoT、实时看板 | 蓝色科技 | 稳重专业，数字感强 |
| 大屏驾驶舱、数据分析、长时间盯屏、BI 报表 | 暗黑专业 | 暗色沉浸，高对比，护眼 |
| 财务对账、合规审批、供应商管理、结算中心 | 绿色清简 | 清爽干净，信任感强 |
| 高端 B 端演示、采购决策、管理层汇报、招商平台 | 紫色商务 | 高级感强，适合对外展示 |

**判断优先级**：用户明确指定 > 场景关键词匹配 > 默认橙白经典

## 使用方式

用户在请求时说明主题即可，例如：

- "帮我做一个**蓝色科技**风格的补货看板"
- "用**暗黑专业**主题做订单列表页"
- 不说主题 → 根据场景自动推荐，**等用户确认后再生成代码**

生成页面时：
1. 在 `App.jsx` 的 `ConfigProvider` 按对应主题配置 `theme` 属性
2. 侧边栏 `SideMenu` 的背景色、文字色按主题规范调整（通过内联样式或 CSS 变量）
3. 标签栏激活色跟随主题主色
4. `code-style.md` 中的主题色 `#FF6B00` 替换为当前主题的 `colorPrimary`
